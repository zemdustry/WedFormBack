# WedForm

## Stack:

### Backend: ⚙️
* **Java version :** Java 17
* **JDK :** OpenJDK 17.0.6
* **Spring:** SpringBoot 3.0.4
* **Maven :** 3.8.6
* **Database :** PostgreSQL


### Frontend: 🖥️ 
* **Angular2+:** 14.2
* **CSS3 + Scss**
* **Bootstrap:** 5.1.3
* **NodeJS:** 18.15.0

### Tools: 🧰 
Vscode, IntelliJ Idea, Docker, Git
